# Beginners, read this!

You need to have a basic understanding of JavaScript and HTML in order to create programs for Windows 93. As a best practice, use JSDoc when writing JavaScript. The guide will walk you through how to create an app.  

**Your best resource is looking at the builtin apps.**  Send a message in the [discord](https://discord.gg/bYFhRTMmMK) if you need help.
