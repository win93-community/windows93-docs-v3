## Copy a Path
You can copy a path in Windows93 by using the `sys42.actions.copyPath` method.

| Property | Type   | Description |
|----------|--------|-------------|
| path     | String | Path to copy |

[View Source](https://v3.windows93.xyz/c/sys/os/actions.js)

## Create File
You can create a file in Windows93 by using the `sys42.actions.createFile` method.

| Property  | Type    | Description                     |
|-----------|---------|---------------------------------|
| path      | String  | Path where the file is created. |
| untitled  | Boolean | Whether the file is untitled.   |

[View Source](https://v3.windows93.xyz/c/sys/os/actions.js)
